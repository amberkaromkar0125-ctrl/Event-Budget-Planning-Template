# Event Budget Planning Template

This repository contains an Excel spreadsheet for managing and tracking event budgets.

## Features
- Estimate quantities and costs for event expenses.
- Record actual quantities and expenses.
- Compare estimated vs. actual costs.
- Organize expenses into different categories.
- Easy-to-use Excel format for budget management.

## Use Cases
- College events
- Corporate events
- Weddings
- Conferences
- Community programs

## File Included
- `Gantt-Chart (1).xlsx` – Event budget planning and expense tracking spreadsheet.
